# Skills LCD — set « Exploitation »

Pack de 5 skills Claude pour exploiter et optimiser une location courte durée.
**Lien privé réservé aux clients — ne pas diffuser.**

## Installation (Claude Desktop / Cowork / claude.ai)

1. Ouvrir le menu **Customize** (barre latérale) → onglet **Plugins**
2. Section « Personal plugins » → bouton **+** → **Add from a repository**
3. Coller l'URL de ce dépôt, puis installer **skills-lcd**

En ligne de commande (Claude Code) :

```
/plugin marketplace add Mishoko39/skills-lcd-x7k2m9
/plugin install skills-lcd@maxxweb-lcd
```

## Les 5 skills

| Skill | Rôle |
|---|---|
| `mon-logement` | Fiche d'identité du logement (à lancer en premier) — PDF de collecte + interview |
| `pricing-saisonnier` | Grille de prix 12 mois + point mort, modes CRÉATION / AJUSTEMENT |
| `fiches-voyageurs` | Kit d'accueil imprimable 5 pièces (guide, WiFi/QR, sortie, équipements, livret local) |
| `reponses-voyageurs-avis` | Réponses aux avis & messages, règles plateformes, exemples avant/après |
| `pilotage-dashboard` | Rapport mensuel HTML (CA, occupation, ADR), tendances 12 mois |
